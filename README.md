# PhoneBook Pro 📱

یک سیستم مدیریت مخاطبین هوشمند با قابلیت استفاده به صورت **خط فرمان (CLI)** و **رابط کاربری گرافیکی (GUI)**. این پروژه با پایتون توسعه یافته و داده‌ها را به صورت پایدار در فایل JSON ذخیره می‌کند.

## 🚀 ویژگی‌ها
*   **پشتیبانی دوگانه:** دارای محیط ساده CLI برای کارهای سریع و محیط گرافیکی جذاب (CustomTkinter) برای مدیریت آسان.
*   **مدیریت کامل:** قابلیت افزودن، ویرایش، حذف و جستجوی مخاطبین.
*   **ذخیره‌سازی محلی:** استفاده از فایل `phone-book.json` برای حفظ داده‌ها بدون نیاز به دیتابیس‌های پیچیده.

## 📂 ساختار پروژه
```text
phonebook_project/
├── main.py            # رابط خط فرمان (CLI)
├── new_ui.py          # رابط کاربری گرافیکی (GUI)
├── functions.py       # منطق اصلی و مدیریت فایل‌ها
└── phone-book.json    # دیتابیس مخاطبین
```

## 🛠 پیش‌نیازها
برای اجرای پروژه، باید پایتون ۳.۸ یا بالاتر داشته باشید. همچنین برای بخش گرافیکی نیاز به نصب کتابخانه `customtkinter` دارید:

```bash
pip install customtkinter
```

## 💻 نحوه استفاده

### ۱. اجرای رابط گرافیکی (GUI)
برای استفاده از محیط گرافیکی، فایل `new_ui.py` را اجرا کنید:
```bash
python new_ui.py
```

### ۲. اجرای رابط خط فرمان (CLI)
برای استفاده از محیط متنی، فایل `main.py` را اجرا کنید:
```bash
python main.py
```

## 🔧 نحوه توسعه
این پروژه از `functions.py` به عنوان هسته اصلی (Backend) استفاده می‌کند. تمامی عملیات خواندن و نوشتن فایل JSON در این فایل متمرکز شده است، بنابراین می‌توانید به راحتی قابلیت‌های جدیدی به آن اضافه کنید.
